## Uma landing page simples feita no next js, com framer motion, tailwind...

## Para começar

### Primeiro baixe as dependencias:

```bash
npm install
```

### Depois, rode o projeto:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### Abrir no browser [http://localhost:3000](http://localhost:3000).
